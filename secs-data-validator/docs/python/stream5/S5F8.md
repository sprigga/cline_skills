# S5F8 — List Enabled Alarm Data

**Direction:** Sent by Equipment Only  
**Reply to:** S5F7R

## Format

```
{L:n
  {L:3
    ALCD   B:1     alarm code byte, >= 128 alarm is set
    ALID   U4:1    Alarm type ID
    ALTX   A:120   alarm text
  }
}
```
