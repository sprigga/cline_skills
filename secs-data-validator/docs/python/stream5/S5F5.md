# S5F5R — List Alarms Request

**Direction:** Sent by Host Only  
**Reply:** S5F6

## Comment

Host sends zero length item for all, otherwise ALID vector.

## Format

```
ALIDVECTOR   (list)U4:n   alarm ID vector
```
