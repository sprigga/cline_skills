# S5F3 — Enable/Disable Alarm Send

**Direction:** Sent by Host Only  
**Reply:** S5F4

## Format

```
{L:2
  ALED   B:1    enable/disable alarm, 128 means enable, 0 disable
  ALID   U4:1   Alarm type ID
}
```
