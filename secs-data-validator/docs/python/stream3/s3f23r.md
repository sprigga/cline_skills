# S3F23R - Port Group Action Req

**Direction:** Sent by Host Only

Comment: not documented in E87, port group names - is this vendor specific?

Format:

{L:3
[PGRPACTION](http://hume.com/secs/items.html#PGRPACTION)

[PORTGRPNAME](http://hume.com/secs/items.html#PORTGRPNAME)

{L:m

{L:2

[PARAMNAME](http://hume.com/secs/items.html#PARAMNAME)

[PARAMVAL](http://hume.com/secs/items.html#PARAMVAL)

 }

} }

        pylist0 = lsplit(TSN_data)
