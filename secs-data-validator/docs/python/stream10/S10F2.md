# S10F2 - Terminal Request Acknowledge

| Message | Description | Direction |
|---------|-------------|-----------|
| **S10F2** | Terminal Request Acknowledge | Sent by Host Only |

## Format

```
ACKC10  - B:1 (always)  acknowledge code
```
