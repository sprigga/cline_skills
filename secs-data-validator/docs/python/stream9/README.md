# SECS-II Stream 9 (S9) - Error Messages

Stream 9 messages are error messages sent by equipment to report communication problems.

## Functions

| Function | Description | Direction |
|----------|-------------|-----------|
| [S9F1](s9f1.md) | Unknown Device ID | Sent by Equipment Only |
| [S9F3](s9f3.md) | Unknown Stream | Sent by Equipment Only |
| [S9F5](s9f5.md) | Unknown Function | Sent by Equipment Only |
| [S9F7](s9f7.md) | Illegal Data | Sent by Equipment Only |
| [S9F9](s9f9.md) | Transaction Timeout | Sent by Equipment Only |
| [S9F11](s9f11.md) | Data Too Long | Sent by Equipment Only |
| [S9F13](s9f13.md) | Conversation Timeout | Sent by Equipment Only |
