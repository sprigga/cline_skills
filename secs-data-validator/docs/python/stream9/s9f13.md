# S9F13 - Conversation Timeout

**Direction:** Sent by Equipment Only

**Format:**
```
{L:2
  MEXP
  EDID }
```

- `MEXP` A:6 — message expected in form of SxxFyy
- `EDID` A:80 — expected data identification (PPID, SPID, or PTN)
