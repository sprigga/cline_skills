# S7F4 - Process Program Send Acknowledge

**Direction:** Sent by Host and Equipment

---

| **S7F4** | Process Program Send Acknowledge | Sent by Host and Equipment |

Comment: Reply to S7F3R. Contains the S7 acknowledge code.

### Format (TSN)

```
[ACKC7]
```

**Format Explanation:**
- **`[ACKC7]`** — S7 Acknowledge Code，型別為 `B:1`（Binary，固定 1 byte）。用來回覆 S7F3R（Process Program Send）的處理結果。

### ACKC7 常見值

| 值 | 意義 |
|----|------|
| 0 | OK — 已接受/完成 |
| 1 | 已存在，PPID 重複 |
| 2 | 空間不足，無法儲存 |
| 3-8 | 其他錯誤（依實裝定義） |
| 9+ | 廠商自訂錯誤碼 |

### C# 解析程式

以下 C# 程式碼是**發送端**接收 S7F4 回覆後的解析邏輯，使用 `secs4net` 函式庫：

### 處理流程

1. **發送請求** — 先以 `sp.SendSecsMsg(7, 3, true, sendData, true)` 發送 S7F3R，並等待回覆。
2. **檢查回傳碼** — `rcr.rc != 0` 表示通訊層面失敗（如 timeout），需中斷處理。
3. **解析 ACKC7** — 用 `SecsPort.ListSplit()` 將回覆資料拆解，驗證格式為 `B:1`，再用 `sp.BinToInt()` 將二進位值轉為整數。
4. **判斷結果** — 根據 ACKC7 的值判斷製程程式是否成功儲存。

```csharp
    // S7F4 C# Parse reply
    // sp is a SecsPort or SecsHost object reference
    RcResult rcr = sp.SendSecsMsg(7, 3, true, sendData, true);
    string TSN_data;
    bool ok=true;
    while (ok) {   // break out of loop on error
        if (rcr.rc != 0) {ok=false; break;}
        TSN_data = rcr.result;
        // expect ACKC7
        // variables for data items and parsing
        int ACKC7;  	// B:1 (always)  S7 acknowledge code
        string [] list0;
        list0 = SecsPort.ListSplit(TSN_data);
        if (list0.Length < 1) { ok = false; break; }
        if (list0[0] != "B:1") { ok = false; break; }
        ACKC7 = sp.BinToInt(list0[1]);
        return;  // parsed ok
        } // end while(ok)
```
