# S7F18 - Delete Process Program Acknowledge

**Direction:** Equipment Only

---

| **S7F18** | Delete Process Program Acknowledge | Equipment Only |

Comment: Reply to S7F17R. Acknowledges the deletion of process programs.

### Format (TSN)

```
[ACKC7]
```

**Format Explanation:**
- **`[ACKC7]`** — S7 Acknowledge Code，型別為 `B:1`（Binary，固定 1 byte）。用來回覆 S7F17R（Delete Process Program Send）的處理結果。

### ACKC7 常見值

| 值 | 意義 |
|----|------|
| 0 | OK — 刪除成功 |
| 1 | PPID 不存在 |
| 2 | 刪除失敗（程式正在使用中） |
| 3-8 | 其他錯誤 |
| 9+ | 廠商自訂錯誤碼 |

### C# 解析程式

以下 C# 程式碼是**發送端 (Host)** 接收 S7F18 回覆後的解析邏輯，使用 `secs4net` 函式庫：

### 處理流程

1. **發送請求** — 先以 `sp.SendSecsMsg(7, 17, true, sendData, true)` 發送 S7F17R，並等待回覆。
2. **檢查回傳碼** — `rcr.rc != 0` 表示通訊層面失敗，需中斷處理。
3. **解析 ACKC7** — 用 `SecsPort.ListSplit()` 將回覆資料拆解，驗證格式為 `B:1`，再用 `sp.BinToInt()` 將二進位值轉為整數。
4. **判斷結果** — 根據 ACKC7 的值判斷刪除操作是否成功。

```csharp
    // S7F18 C# Parse reply
    SecsHost sp=(SecsHost)your_SecsHost;
    RcResult rcr = sp.SendSecsMsg(7, 17, true, sendData, true);
    string TSN_data;
    bool ok=true;
    while (ok) {   // break out of loop on error
        if (rcr.rc != 0) {ok=false; break;}
        TSN_data = rcr.result;
        // expect ACKC7
        // variables for data items and parsing
        int ACKC7;  	// B:1 (always)  S7 acknowledge code
        string [] list0;
        list0 = SecsPort.ListSplit(TSN_data);
        if (list0.Length < 1) { ok = false; break; }
        if (list0[0] != "B:1") { ok = false; break; }
        ACKC7 = sp.BinToInt(list0[1]);
        return;  // parsed ok
        } // end while(ok)
```
