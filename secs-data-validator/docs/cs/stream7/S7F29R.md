# S7F29R - Process Program Verification Inquire

**Source:** [SECS-II Automated Code Generation Tool](http://hume.com/secs/msgsCS.html)

**Sent by:** Equipment Only

**Comment:** This is a multiblock inquire for S7F27 so the LENGTH is the S7F27 message length and not the PP length.

## Format

[LENGTH](http://hume.com/secs/items.html#LENGTH)

## Details

Request from equipment to inquire if the host will accept a process program verification message (S7F27). The LENGTH is the S7F27 message length, not the process program length.
