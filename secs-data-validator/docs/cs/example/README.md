# SECS-II 實務應用範例

本目錄收錄跨多支訊息的端到端實務情境，補充各訊息文件（`stream*/`）中的單一訊息說明。

每份文件聚焦在「一個完整工作流」，包含：訊息序列圖、前置作業、secs4net C# 實作、錯誤處理策略。

---

## 範例清單

| 文件 | 涵蓋訊息 | 情境說明 |
|------|----------|----------|
| [S6F3R_S6F4_example.md](S6F3R_S6F4_example.md) | S6F3R, S6F4 | 設備主動送離散變數資料（事件觸發量測回報）|
| [S2F15R_S2F16_example.md](S2F15R_S2F16_example.md) | S2F15R, S2F16, S2F13R, S2F29R | Host 查詢並設定設備工藝參數（EC 管理工作流）|

---

## 與各 Stream 文件的關係

```
docs/secs/cs/
├── stream2/          ← 單一訊息格式與 parse 說明
├── stream6/          ← 單一訊息格式與 parse 說明
└── example/          ← 跨訊息端到端情境（本目錄）
```

各 stream 目錄的文件說明「這支訊息長什麼樣、怎麼 parse」；本目錄說明「這些訊息在實際連線流程中如何組合使用」。
