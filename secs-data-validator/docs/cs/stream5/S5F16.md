# S5F16 - Exception Recovery Complete Confirm

**Direction:** Sent by Host Only

---

Format:

_header only_

```csharp
    // S5F16 C# Parse reply
    SecsPort sp=(SecsPort)your_SecsPort
    RcResult rcr = sp.SendSecsMsg(5, 15, true, sendData, true);
    string TSN_data;
    bool ok=true;
    while (ok) {   // break out of loop on error
        if (rcr.rc != 0) {ok=false; break;}
        TSN_data = rcr.result;
        // no data expected
        return;  // parsed ok
        } // end while(ok)
```
