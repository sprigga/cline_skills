# S10F4 - Terminal Display, Single Acknowledge

**Direction:** Sent by Equipment Only

---

## Format 說明

```
ACKC10
```

| 欄位 | 型別 | 說明 |
|------|------|------|
| [`ACKC10`](http://hume.com/secs/items.html#ACKC10) | `B:1`（固定） | Acknowledge Code，確認碼，以 binary 1 byte 表示 |

- 此訊息為單一資料項目，無 List 包裝
- `ACKC10 = 0` 表示成功接受；非零值表示錯誤
- 此訊息是對 S10F3（Terminal Display, Single）的 reply

---

## Codebase 說明（C# / dmhclient）

| 方法 / 變數 | 說明 |
|-------------|------|
| `sp.SendSecsMsg(10, 3, true, sendData, true)` | Host 發送 S10F3 並等待 S10F4 reply |
| `rcr.rc != 0` | 檢查通訊層是否成功（`rc == 0` 表示正常） |
| `rcr.result` | 取得回傳的 TSN 資料字串 |
| `SecsPort.ListSplit(TSN_data)` | 將 TSN 字串解析為字串陣列；`[0]` 為型別標籤，`[1]` 為資料值 |
| `list0[0] != "B:1"` | 驗證 ACKC10 欄位型別為 binary 1 byte |
| `sp.BinToInt(list0[1])` | 將 binary 字串轉換為整數（取得 ACKC10 數值） |

```csharp
    // S10F4 C# Parse reply
    SecsHost sp=(SecsHost)your_SecsHost;
    RcResult rcr = sp.SendSecsMsg(10, 3, true, sendData, true);
    string TSN_data;
    bool ok=true;
    while (ok) {   // break out of loop on error
        if (rcr.rc != 0) {ok=false; break;}
        TSN_data = rcr.result;
        // expect ACKC10
        // variables for data items and parsing
        int ACKC10;  	// B:1 (always)  acknowledge code
        string [] list0;
        list0 = SecsPort.ListSplit(TSN_data);
        if (list0.Length < 1) { ok = false; break; }
        if (list0[0] != "B:1") { ok = false; break; }
        ACKC10 = sp.BinToInt(list0[1]);
        return;  // parsed ok
        } // end while(ok)
```
