# S6F10 - Formatted Variable Ack

**Direction:** Sent by Host Only

---

## Format

**Message Structure:** Host acknowledges receipt of formatted variable data (S6F9R) from equipment.

### Format Explanation

The **Format** shows a simple single-item message:

```
[ACKC6]
```

- **`[ACKC6]`** — Acknowledge Code (B:1) — a single binary byte
  - `0` = OK (data accepted)
  - Non-zero = Error/rejection code

### SECS Notation Reference
- `[ITEMID]` = A single data item
- `B:1` = Binary item, 1 byte

---

## Codebase

**Message Type:** Reply (sent in response to S6F9R)

### Codebase Explanation

This C# code demonstrates how to **send and parse a reply** to S6F9R (S6F10) using the **secs4net** library.

#### Key Components

1. **Sending the Message** (line 14)
   - `sp.SendSecsMsg(6, 9, ...)` sends S6F9R to the equipment and waits for the reply (S6F10)
   - Returns `RcResult` containing the response in `rcr.result`

2. **Handler Signature** (lines 16-19)
   - This is **reply parsing**, not a receive handler
   - `rcr.rc` = Return code (0 = success)
   - `TSN_data` = The reply message data as a string

3. **Parsing Logic**
   - **Line 24**: `ListSplit(TSN_data)` breaks the message into elements:
     - `[0]` = "B:1" (binary type, 1 byte)
     - `[1]` = The actual value (binary string)

   - **Line 27**: `sp.BinToInt(list0[1])` converts the binary value to integer
     - Typically 0 for OK, non-zero for error

#### Parsing Workflow
1. Send the message with `SendSecsMsg()` and wait for reply
2. Check return code `rcr.rc` for communication errors
3. Split reply data → check first element is "B:1"
4. Convert binary to integer for the acknowledge code

#### Common Acknowledge Codes
| Code | Meaning |
|------|---------|
| 0 | OK — data accepted |
| 1 | BUSY — equipment busy, retry later |
| 2 | DENIED — equipment denied the request |

```
    // S6F10 C# Parse reply
    SecsPort sp=(SecsPort)your_SecsPort
    RcResult rcr = sp.SendSecsMsg(6, 9, true, sendData, true);
    string TSN_data;
    bool ok=true;
    while (ok) {   // break out of loop on error
        if (rcr.rc != 0) {ok=false; break;}
        TSN_data = rcr.result;
        // expect ACKC6
        // variables for data items and parsing
        int ACKC6;  	// B:1 (always)  acknowledge code, 0 ok
        string [] list0;
        list0 = SecsPort.ListSplit(TSN_data);
        if (list0.Length < 1) { ok = false; break; }
        if (list0[0] != "B:1") { ok = false; break; }
        ACKC6 = sp.BinToInt(list0[1]);
        return;  // parsed ok
        } // end while(ok)
```
