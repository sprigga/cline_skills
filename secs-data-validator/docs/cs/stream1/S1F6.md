# S1F6 - Formatted Status Data

**Direction:** Sent by Equipment Only

---

| **S1F6** | Formatted Status Data | Sent by Equipment Only |

Comment: message structure varies, superseded by dynamic reports

Format:

{L:n
[SV](http://hume.com/secs/items.html#SV)}

```
    // S1F6 C# Parse reply
    SecsHost sp=(SecsHost)your_SecsHost;
    RcResult rcr = sp.SendSecsMsg(1, 5, true, sendData, true);
    string TSN_data;
    bool ok=true;
    while (ok) {   // break out of loop on error
        if (rcr.rc != 0) {ok=false; break;}
        TSN_data = rcr.result;
        // expect L:n SV
        // variables for data items and parsing
        string SV;  	// A:n (varies)  status variable value
        string [] list0;
        list0 = SecsPort.ListSplit(TSN_data);
        if (list0.Length < 1) { ok=false; break; }
        int n;
        for(n=1; n < list0.Length; n++) {
            string [] list1;
            list1 = SecsPort.ListSplit(list0[n]);
            if (list1.Length < 1) { ok = false; break; }
            if (list1.Length == 2) { SV = list1[1];} else { SV = ""; }
            }
        if (!ok) break;
        return;  // parsed ok
        } // end while(ok)
```
