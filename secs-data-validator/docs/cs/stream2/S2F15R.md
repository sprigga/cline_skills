# S2F15R - New Equipment Constant Send

**Direction:** Host Only

---

| **S2F15R** | New Equipment Constant Send | Host Only |

Comment: Constant is a misnomer, they are settable parameters.

Format:

{L:n
{L:2

[ECID](http://hume.com/secs/items.html#ECID)

[ECV](http://hume.com/secs/items.html#ECV)

 }}

## Format 說明

| 層級 | 結構 | 說明 |
|------|------|------|
| 外層 | `L:n` | 包含 n 筆常數設定；n=0 表示不設定任何常數 |
| 內層 | `L:2` | 每筆固定 2 個元素：ECID 和 ECV |
| `ECID` | `U4:1`（GEM 規定） | Equipment Constant ID，設備常數的識別碼 |
| `ECV` | 任何純量型別 (`A`, `U1`–`U8`, `I1`–`I8`, `F4`, `F8` 等) | Equipment Constant Value，常數的設定值 |

> 備註：名稱雖叫「Constant」，實際上是可在運行時設定的可變參數。

```csharp
// S2F15R C# Receive message - add next line to setup
// SecsPort sp = your_SecsPort;
sp.MessageTypeAdd(2, 15, new SecsMsgReceiveDelegate(recv_S2F15R));

// receive method
void recv_S2F15R(object sender, int stream, int function, bool send_reply, int transID, string TSN_data, string header) {
    SecsPort sp = (SecsPort)sender;
    bool ok = true;
    while (ok) {   // break out of loop on error

        // 預期格式：L:n { L:2 ECID ECV }
        // 變數宣告
        string ECID;  // U4:1（GEM 規定必須是 U4）裝置常數識別碼
        string ECV;   // 任何純量型別（A, U1-U8, I1-I8, F4, F8 等），常數設定值

        // ── 步驟 1：解析外層 L:n ──
        string[] list0 = SecsPort.ListSplit(TSN_data);
        // list0[0] = "L:n"，list0[1..n] 為每筆 {L:2 ECID ECV}
        if (list0.Length < 1) { ok = false; break; }

        // ── 步驟 2：逐筆解析每組 {L:2 ECID ECV} ──
        for (int n = 1; n < list0.Length; n++) {

            // 解析內層 L:2
            string[] list1 = SecsPort.ListSplit(list0[n]);
            // list1[0]="L:2"，list1[1]=ECID token，list1[2]=ECV token
            if (list1.Length != 3 || list1[0] != "L:2") { ok = false; break; }

            // 解析 ECID（U4:1）
            string[] list2 = SecsPort.ListSplit(list1[1]);
            if (list2.Length < 1) { ok = false; break; }
            ECID = (list2.Length == 2) ? list2[1] : "";  // list2[0]=型別 token，list2[1]=值

            // 解析 ECV（任意純量型別）
            list2 = SecsPort.ListSplit(list1[2]);
            if (list2.Length < 1) { ok = false; break; }
            ECV = (list2.Length == 2) ? list2[1] : "";   // list2[0]=型別 token，list2[1]=值

            // TODO: 在此處套用 ECID/ECV 至設備參數
        }
        if (!ok) break;

        // ── 步驟 3：發送回覆 S2F16（若對方要求回覆）──
        if (send_reply) {
            string reply; // TODO: 建立 S2F16 回覆資料（EAC - Equipment Acknowledge Code）
            sp.SendReply(2, 16, transID, reply);
        }
        return;

    } // end while(ok)

    // 解析失敗：發送 S9F7（非法資料錯誤）
    sp.SendS9(7, header);
} // end recv_S2F15R
```

## 代碼說明

| 步驟 | 代碼 | 說明 |
|------|------|------|
| 登錄處理器 | `sp.MessageTypeAdd(2, 15, ...)` | 向 SecsPort 登錄 S2F15 的接收回呼 |
| 解析外層 list | `ListSplit(TSN_data)` → `list0` | `list0[0]`=`"L:n"`，`list0[1..n]` 為各筆常數資料 |
| 遍歷各筆 | `for n=1; n < list0.Length` | 逐一處理每組 `{L:2 ECID ECV}` |
| 驗證 L:2 結構 | `list1.Length != 3 \|\| list1[0] != "L:2"` | Length=3 是因為：`"L:2"` + ECID token + ECV token |
| 解析 ECID | `list2 = ListSplit(list1[1])` | `list2[0]`=型別（如 `"U4:1"`），`list2[1]`=數值字串 |
| 解析 ECV | `list2 = ListSplit(list1[2])` | `list2[0]`=型別，`list2[1]`=值；空 list 時給空字串 |
| 成功回覆 | `sp.SendReply(2, 16, transID, reply)` | 發送 S2F16（EAC，設備確認碼），需填入回覆資料 |
| 錯誤處理 | `sp.SendS9(7, header)` | 資料格式錯誤時發送 S9F7（Illegal Data） |
