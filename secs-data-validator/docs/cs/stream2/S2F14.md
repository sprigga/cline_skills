# S2F14 - Equipment Constant Data

**Direction:** Equipment Only

---

| **S2F14** | Equipment Constant Data | Equipment Only |

Comment: An L:0 is substituted if an ECID does not exist, L format is not allowed for an ECV.

Format:

```
{L:n
  [ECV]
}
```

**Format Explanation:**
- `{L:n}` — A SECS List structure with `n` elements
- `[ECV]` — Equipment Constant Value; can be any scalar SECS data type (A, B, JIS, U, I, F, etc.)
  - Returned as a list element for each equipment constant
  - If an ECID (Equipment Constant ID) does not exist, an empty list (L:0) is substituted

### C# Parser Example

```csharp
// S2F14 C# Parse reply
SecsHost sp=(SecsHost)your_SecsHost;
RcResult rcr = sp.SendSecsMsg(2, 13, true, sendData, true);
string TSN_data;
bool ok=true;
while (ok) {   // break out of loop on error
    if (rcr.rc != 0) {ok=false; break;}
    TSN_data = rcr.result;
    
    // Expect L:n where each element contains ECV
    // Step 1: Split the outer list structure
    string [] list0;
    list0 = SecsPort.ListSplit(TSN_data);
    if (list0.Length < 1) { ok=false; break; }
    
    // Step 2: Iterate through each list element
    int n;
    for(n=1; n < list0.Length; n++) {
        // Step 3: Split each nested list element
        string [] list1;
        list1 = SecsPort.ListSplit(list0[n]);
        if (list1.Length < 1) { ok = false; break; }
        
        // Step 4: Extract ECV (equipment constant value)
        // Can be any scalar type; check list1[1] for the actual value
        string ECV;
        if (list1.Length == 2) { 
            ECV = list1[1];  // ECV found
        } else { 
            ECV = "";  // No ECV at expected position
        }
    }
    if (!ok) break;
    return;  // Parsing completed successfully
} // end while(ok)
```

**Code Flow Explanation:**
1. **Send Request** — S2F13 (request) triggers S2F14 (reply) from equipment
2. **Parse Outer List** — `ListSplit(TSN_data)` breaks the L:n structure into list0 elements
3. **Loop Through Elements** — Each element (list0[n]) is processed individually
4. **Parse Nested List** — `ListSplit(list0[n])` extracts fields from each list element
5. **Extract Value** — `list1[1]` contains the actual equipment constant value (ECV)
6. **Error Handling** — Exit loop on parsing error; return on success
