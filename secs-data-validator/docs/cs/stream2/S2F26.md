# S2F26 - Loopback Diagnostic Data

**Direction:** Host and Equipment

---

| **S2F26** | Loopback Diagnostic Data | Host and Equipment |

## Format

**Message Body:** [ABS](http://hume.com/secs/items.html#ABS) — a binary string containing loopback diagnostic data

**ABS** (Any Binary String) is a SECS-II data type that represents arbitrary binary data. In S2F26, it carries the diagnostic data payload that is echoed back in a loopback test.

## C# Code Example

```csharp
// S2F26 — Parse loopback diagnostic reply
// sp: SecsPort or SecsHost connection object (SECS4NET library)
// 
// This code sends S2F25 (loopback request with data) and parses the S2F26 reply,
// extracting the echoed binary diagnostic data.

RcResult rcr = sp.SendSecsMsg(2, 25, true, sendData, true);
string TSN_data;
bool ok = true;

while (ok) {   // break out of loop on error
    if (rcr.rc != 0) { ok = false; break; }
    TSN_data = rcr.result;
    
    // Expected data structure: List containing one ABS item
    // Parse the TSN (SECS data) string into components
    string[] list0;
    list0 = SecsPort.ListSplit(TSN_data);
    
    // Validation: must have at least 1 element
    if (list0.Length < 1) { ok = false; break; }
    
    // Validation: first element must be a binary string (B:n format)
    if (!list0[0].StartsWith("B:")) { ok = false; break; }
    
    // Extract binary values from remaining elements
    int[] ABS = new int[list0.Length - 1];
    for (int i = 0; i < ABS.Length; i++) {
        ABS[i] = sp.BinToInt(list0[1 + i]);  // Convert SECS binary format to integer
    }
    
    return;  // parsed successfully
}
```

### Code Breakdown

- **SendSecsMsg(2, 25, true, sendData, true):** Sends S2F25 loopback request and waits for S2F26 reply
  - Parameters: stream=2, function=25, wait_for_reply=true, payload=sendData, alloc_SDACK=true
- **SecsPort.ListSplit():** Splits SECS list structure string into components
- **BinToInt():** Converts SECS binary format to integer value
- **ABS array:** Holds the parsed loopback diagnostic bytes
