# Demo Directory

This directory contains generated output files from the SECS-II SML Format Validator skill.

## Directory Structure

- **`python/`** - Python scripts and code examples
- **`csharp/`** - C# code and handler implementations  
- **`logs/`** - Processed and analyzed log files
- **`templates/`** - SML template files and examples
- **`reports/`** - Validation reports and analysis outputs
- **`docs/`** - Generated documentation

## Purpose

All code, text files, and output generated by the secs-data-validator skill are organized in this directory structure to maintain clean separation of different file types and make it easy to locate specific types of outputs.
