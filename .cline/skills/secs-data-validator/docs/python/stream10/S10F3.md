# S10F3 - Terminal Display, Single

| Message | Description | Direction |
|---------|-------------|-----------|
| **S10F3[R]** | Terminal Display, Single | Sent by Host Only |

## Format

```
{L:2
  TID   - B:1 (always)  terminal ID
  TEXT  - A:120 (varies)  line of text for display, no standard max size
}
```
