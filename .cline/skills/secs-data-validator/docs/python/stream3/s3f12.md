# S3F12 - Matl ID Request Ack

**Direction:** Sent by Host Only

Format:

{L:3
[PTN](http://hume.com/secs/items.html#PTN)

[MIDRA](http://hume.com/secs/items.html#MIDRA)

[MID](http://hume.com/secs/items.html#MID) }
