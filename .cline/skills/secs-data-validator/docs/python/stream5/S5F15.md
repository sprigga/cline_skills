# S5F15 — Exception Recovery Complete Notify

**Direction:** Sent by Equipment Only  
**Reply:** S5F16  
**Comment:** L:2* can be L:2 or L:0; see E41

## Format

```
{L:3
  TIMESTAMP   A:32   ECV TimeFormat controls format
  EXID        A:20   exception identifier
  {L:2
    ACKA    TF:1   request success
    {L:2*
      ERRCODE   U4:1   error code
      ERRTEXT   A:80   description of ERRCODE
    }
  }
}
```
