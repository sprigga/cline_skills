# S5F11 — Exception Clear Notify

**Direction:** Sent by Equipment Only  
**Reply:** S5F12  
**Comment:** see E41

## Format

```
{L:4
  TIMESTAMP    A:32   ECV TimeFormat controls format
  EXID         A:20   exception identifier
  EXTYPE       A:5    exception type, "ALARM" or "ERROR"
  EXMESSAGE    A:n    exception description
}
```
