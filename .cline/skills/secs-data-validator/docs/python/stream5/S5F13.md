# S5F13R — Exception Recover Request

**Direction:** Sent by Host Only  
**Reply:** S5F14  
**Comment:** see E41

## Format

```
{L:2
  EXID      A:20   exception identifier
  EXRECVRA  A:40   exception recovery action description
}
```
