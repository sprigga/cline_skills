# S1F12 - Status Variable Namelist Reply

**Direction:** Sent by Equipment Only

**Comment:** A:0 for SVNAME and UNITS indicates unknown SVID

**Format:** `{L:n {L:3 SVID SVNAME UNITS`

## Python Code
