# S1F13R - Establish Communications Request

**Direction:** Different Host and Equipment Use

**Equipment Format:** `{L:2 MDLN SOFTREV`

**Host Format:** `L:0`

        if len(pylist0) != 3 or pylist0[0] != "L:2":
