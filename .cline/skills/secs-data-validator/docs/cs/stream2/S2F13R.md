# S2F13R - Equipment Constant Request

**Direction:** Host Only

---

| **S2F13R** | Equipment Constant Request | Host Only |

Comment: Host sends L:0 to receive all in predefined order

## Format (TSN)

{L:n
[ECID](http://hume.com/secs/items.html#ECID)}

- **`L:n`** — 一個 list 容器，包含 `n` 個元素。`n` 可以是 `0`（即 `L:0`），代表**請求回傳所有設備常數**（以預定義順序）。
- **`[ECID]`** — `ECID` (Equipment Constant ID) 是設備常數的 ID，用方括號 `[...]` 表示為**選填**（可出現 0 次或多次）。當 `n=0` 時，host 不帶任何 ECID，裝備端回傳全部常數。

### 範例

| 傳送內容 | 意義 |
|---------|------|
| `L:0` | 請求回傳所有設備常數（空 list，不指定任何 ECID） |
| `L:2 {U4:1} {U4:5}` | list 有 2 個元素，請求回傳 ECID=1 和 ECID=5 的常數值 |

## C# 接收處理程式

以下 C# 程式碼是**設備端 (Equipment)** 接收 S2F13R 後的處理邏輯，使用 `secs4net` 函式庫：

### 處理流程

1. **註冊 handler** — 在初始化時用 `sp.MessageTypeAdd(2, 13, recv_S2F13R)` 將回呼函式綁定到 Stream 2, Function 13。
2. **解析 TSN 資料** — 用 `SecsPort.ListSplit()` 將 TSN 字串拆解為 list 結構，逐筆取出 ECID 值。
3. **回覆 S2F14** — 解析成功後以 `sp.SendReply(2, 14, transID, reply)` 回覆 S2F14（Equipment Constant Send），`reply` 的內容需根據實際需求建構。
4. **錯誤處理** — 若 TSN 資料格式錯誤（list 結構不合法），送出 `sp.SendS9(7, header)`（S9F7 = illegal data）通知 host。

```csharp
// S2F13R C# Receive message - add next line to setup
//SecsPort sp=your_SecsPort;
sp.MessageTypeAdd( 2, 13, new SecsMsgReceiveDelegate(recv_S2F13R));

// receive method
void recv_S2F13R(object sender, int stream, int function, bool send_reply, int transID, string TSN_data, string header) {
    SecsPort sp = (SecsPort)sender;
    bool ok=true;
    while (ok) {   // break out of loop on error
        // expect L:n ECID
        // variables for data items and parsing
        string ECID;  	// U4:1 (varies)  equipment constant ID, GEM requires U4
        string [] list0;
        list0 = SecsPort.ListSplit(TSN_data);
        if (list0.Length < 1) { ok=false; break; }
        int n;
        for(n=1; n < list0.Length; n++) {
            string [] list1;
            list1 = SecsPort.ListSplit(list0[n]);
            if (list1.Length < 1) { ok = false; break; }
            if (list1.Length == 2) { ECID = list1[1];} else { ECID = ""; }
            }
        if (!ok) break;
        if (send_reply) {
            string reply; // TBD create reply
            sp.SendReply(2, 14, transID, reply);
            }
        return;
        } // end while(ok)
    // bad data
    sp.SendS9(7, header);
    } // end recv_S2F13R
```
