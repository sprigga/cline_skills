# S2F25R - Loopback Diagnostic Request

**Direction:** Host and Equipment

---

| **S2F25R** | Loopback Diagnostic Request | Host and Equipment |

Comment: it is useful to test the telnet escape sequence

Format:

[ABS](http://hume.com/secs/items.html#ABS)

## Format（格式）詳細說明

`[ABS]` 表示此消息只包含一個數據元素：

- **ABS** = 任意二進制字符串（Abstract Binary String）
  - SECS-II 標準用 `B:n` 表示，其中 `n` 是位元組數
  - 可以是任何長度的二進制數據（沒有固定長度限制）

**消息用途：**
- 主機向設備發送測試數據（可以是任何二進制內容）
- 設備接收並原樣回覆（S2F26 回復消息）
- 用於檢驗通信通道的完整性和轉義字符的正確性

## Codebase（C# 代碼）詳細說明

### 函數簽名
```csharp
void recv_S2F25R(object sender, int stream, int function, bool send_reply, int transID, string TSN_data, string header)
```

### 函數參數說明

| 參數 | 含義 |
|------|------|
| `sender` | SecsPort 或 SecsHost 連接對象 |
| `stream` | SECS-II Stream 編號（這裡是 2） |
| `function` | 函數編號（這裡是 25） |
| `send_reply` | 是否需要發送回復 |
| `transID` | 交易 ID（用於配對請求和回復） |
| `TSN_data` | 傳輸的數據（SECS 格式字符串） |
| `header` | 消息頭信息 |

### 代碼邏輯分解

**1. 解析 SECS 格式字符串**
```csharp
list0 = SecsPort.ListSplit(TSN_data);
```
將 SECS 格式的字符串拆分成列表元素
- 例如：`TSN_data = "B:4 0x12 0x34 0x56 0x78"` → `list0 = ["B:4", "0x12", "0x34", "0x56", "0x78"]`

**2. 驗證最小數據長度**
```csharp
if (list0.Length < 1) { ok = false; break; }
```
確保至少有一個元素存在

**3. 驗證數據類型**
```csharp
if (!list0[0].StartsWith("B:")) { ok = false; break; }
```
檢驗第一個元素是否為二進制類型（必須以 `B:` 開頭）
- 如果不是，則標記為錯誤並跳出循環

**4. 轉換二進制數據為整數數組**
```csharp
int [] ABS = new int[list0.Length-1];
for(int i=0; i < ABS.Length; i++) {
    ABS[i] = sp.BinToInt(list0[1+i]);
}
```
- `list0[1+i]` 跳過第一個元素（類型標記），取實際的二進制值
- `BinToInt()` 將十六進制字符串轉換為整數

**5. 發送回復消息**
```csharp
if (send_reply) {
    string reply; // TBD create reply
    sp.SendReply(2, 26, transID, reply);
}
```
發送 S2F26 回復消息
- 發送流 2、函數 26 的回復
- 使用相同的 `transID` 將回復與請求配對
- `reply` 通常應包含原樣收到的 ABS 數據

### 處理流程總結

| 階段 | 操作 | 說明 |
|------|------|------|
| **解析** | `ListSplit()` | 將 SECS 字符串分解為元素 |
| **驗證** | 檢查長度 + 檢查類型 | 確保數據符合 ABS（二進制字符串）格式 |
| **轉換** | `BinToInt()` | 將十六進制值轉換為整數 |
| **回復** | `SendReply()` | 發送 S2F26 回復消息 |

            }
        if (send_reply) {
            string reply; // TBD create reply

            }
        return;
        } // end while(ok)
    } // end recv_S2F25R
