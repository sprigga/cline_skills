# SML（SECS Message Language）語法參考

## 訊息基本結構

```
S<stream>F<function> [W]
<item>
.
```

- `S<stream>F<function>`：Stream 1–127，Function 0–255（奇數為主動發送，偶數為回應）
- `W`（可選）：Wait bit，表示發送方等待回應
- `<item>`：訊息本體，單一 data item
- `.`：訊息結束符（必須）

**範例：**
```
S1F1 W
<L
  <A 'HostName'>
  <U2 1234>
>.
```

---

## Data Item Types

### 容器類型

| 類型 | 說明 | 語法 |
|------|------|------|
| `L` | List（有序集合） | `<L [count] item1 item2 ...>` |

### 字串類型

| 類型 | 說明 | 語法 |
|------|------|------|
| `A` | ASCII string | `<A 'text'>` 或 `<A "text">` 或 `<A [len] 'text'>` |
| `J` | JIS-8 string | `<J 'text'>` |

### 布林與二進位

| 類型 | 說明 | 語法 |
|------|------|------|
| `BOOLEAN` | Boolean | `<BOOLEAN T>` 或 `<BOOLEAN F>` |
| `B` | Binary | `<B 0x0A 0xFF ...>` 或 `<B [len] 0x...>` |

### 無號整數

| 類型 | 位元 | 範圍 | 語法 |
|------|------|------|------|
| `U1` | 8-bit | 0–255 | `<U1 value>` |
| `U2` | 16-bit | 0–65535 | `<U2 value>` |
| `U4` | 32-bit | 0–4294967295 | `<U4 value>` |
| `U8` | 64-bit | 0–18446744073709551615 | `<U8 value>` |

### 有號整數

| 類型 | 位元 | 範圍 | 語法 |
|------|------|------|------|
| `I1` | 8-bit | -128–127 | `<I1 value>` |
| `I2` | 16-bit | -32768–32767 | `<I2 value>` |
| `I4` | 32-bit | -2147483648–2147483647 | `<I4 value>` |
| `I8` | 64-bit | 完整有號 64-bit | `<I8 value>` |

### 浮點數

| 類型 | 位元 | 語法 |
|------|------|------|
| `F4` | 32-bit IEEE 754 | `<F4 3.14>` |
| `F8` | 64-bit IEEE 754 | `<F8 3.14159265>` |

## 語法規則

### 1. 括號配對
每個 `<` 必須對應一個 `>`，且必須正確巢狀。

```
# 正確
<L
  <A 'test'>
  <U1 1>
>

# 錯誤：缺少關閉括號
<L
  <A 'test'>
  <U1 1>
```

### 2. 空 Item
允許空的 item：
```
<A>        # 空字串
<L>        # 空 list
<U1>       # 空值（長度 0）
```

### 3. 長度標注（可選）
可在 type 後標注預期長度，用於驗證：
```
<A [8] 'HostName'>   # 標注長度為 8
<U2 [3] 1 2 3>       # 標注 3 個 U2 值
```

### 4. 多值（Array）
數值類型可包含多個值（形成 array）：
```
<U1 1 2 3 4>         # 4 個 U1 值
<F4 1.0 2.0 3.0>     # 3 個 F4 值
```

### 5. 訊息結束符
完整訊息必須以 `.` 結尾（單獨一行或緊接最後 item 後）。

## 常見格式錯誤

| 錯誤類型 | 說明 | 錯誤範例 |
|---------|------|---------|
| 無效 stream/function | SxFy 格式錯誤 | `S0F1`（stream 0 保留）、`S1F256` |
| 未知 item type | 使用不存在的 type | `<X 'test'>` |
| 括號不配對 | `<` 和 `>` 數量不符 | `<L <A 'x'>` |
| 數值超出範圍 | 值超過 type 允許範圍 | `<U1 300>` |
| 缺少訊息結束符 | 訊息沒有 `.` 結尾 | - |
| 字串引號不配對 | 引號未關閉 | `<A 'test>` |
| 無效 Boolean 值 | 只允許 T 或 F | `<BOOLEAN 1>` |
| 無效 Binary 格式 | 非 0x 開頭的 hex | `<B FF>` |

## 常見訊息範例

### S1F1 – Are You There (W)
```
S1F1 W
.
```

### S1F2 – On Line Data
```
S1F2
<L
  <A 'MDLN-001'>
  <A '1.0.0'>
>.
```

### S6F11 – Event Report Send
```
S6F11 W
<L
  <U4 12345>
  <U4 1>
  <L
    <L
      <U4 101>
      <L
        <U4 1001>
        <U1 255>
      >
    >
  >
>.
```

### S2F41 – Host Command Send
```
S2F41 W
<L
  <A 'START'>
  <L
    <L
      <A 'Speed'>
      <U2 100>
    >
  >
>.
```
